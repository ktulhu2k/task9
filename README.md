# Система бронирования авиабилетов

Учебный проект по разработке клиент-серверного приложения для бронирования авиабилетов на основе **FastAPI**, с использованием современных практик:

- REST API с OpenAPI/Swagger;
- Аутентификация и авторизация через **JWT**;
- Хранение данных в **PostgreSQL**;
- Кэширование свободных мест через **Redis**;
- Управление схемой БД через **Alembic**;
- Контейнеризация через **Docker Compose**;

---

##  Функциональность

### Для всех пользователей:
- Получение списка всех рейсов (`GET /api_v1/flights`);
- Поиск рейсов по городам отправления/прибытия и дате (`GET /api_v1/search`);
- Просмотр всех бронирований (`GET /api_v1/bookings`).

### Для авторизованных пользователей:
- Регистрация (`POST /api_v1/register`);
- Вход (`POST /api_v1/login`);
- Бронирование места на рейсе (`POST /api_v1/booking`);
- Удаление собственного бронирования (`DELETE /api_v1/booking/{id}`).


##  Быстрый запуск

### Требования
- Docker и Docker Compose
- Python 3.8+ (для локального запуска)

```bash

# 2. Приложение доступно по адресу:
